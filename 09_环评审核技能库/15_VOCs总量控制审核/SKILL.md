# 15_VOCs总量控制审核

## 1. 审核目标
VOCs总量控制审核的独立审核，判断报告中的相关内容是否满足法规、标准和指南要求。

## 2. 适用场景
- 佛山市塑胶行业建设项目环境影响报告表
- 涉VOCs、胶水/涂胶、复合/熟化工艺的项目

## 3. 不适用场景
- 报告书级别环评文件
- 非塑胶行业项目
- 纯注塑/挤塑项目（无VOCs产生环节，参照使用）

## 4. 输入材料
- 环评报告MinerU解析后的Markdown全文
- 可选：法规库检索结果（从 `03_指南解析_明文标准库/` 获取）
- 对应Dify工作流：`06_Dify工作流/19-废气收集效率判定.yml`

## 5. 文本提取策略

### 关键词评分筛选
| 关键词组 | 权重 |
|---------|------|
| `收集率|收集效率|集气效率|捕集效率|废气收集效率` | 38 |
| `65%|70%|80%|90%|效率` | 24 |
| `广东省工业源挥发性有机物减排量核算方法|减排量核算方法|2023年修订版|表3.3-2` | 38 |
| `集气罩|密闭集气罩|局部密闭|点对点|包围型|外部集气罩|密闭空间` | 32 |
| `VOCs|挥发性有机物|非甲烷总烃|注塑废气|有机废气` | 28 |
| `控制风速|负压|密闭|罩口` | 20 |
| `有组织|无组织|总产生量|产生量|排放量` | 18 |

### 章节边界定位
- **起始标记**: 运营期, 营运期, 废气, 主要环境影响和保护措施
- **结束标记**: 废水自行监测计划, 声环境, 固体废物, 环境保护措施监督检查清单

原始工作流使用关键词评分累积 + 章节边界截断的混合策略，总分超过阈值且位于目标章节内的文本片段被提取为审核素材。

## 6. 法规库检索

审核时需从法规库检索以下模块的标准条款：

| STD_018 | 稀释比说明 | GB 31572-2015 5.2 | 使用稀释剂的工序（涂布/印刷/贴合）须说明稀释比(%)和稀释剂类型。稀释剂VOCs全部计入源强 [依据: GB 31572-2015 5.2] |
| STD_019 | 低VOCs替代论证 | GB 33372-2020 附录A | 使用溶剂型胶粘剂/油墨/清洗剂的，须评估低VOCs替代方案的可行性。若不可替代，须说明技术原因并承诺VOCs排放最小化 [依据: GB 33372-2020 附录A] |
| STD_061 | VOCs总量指标 | 粤环发[2019]2号  | 须明确VOCs有组织+无组织年排放总量。佛山4+2重点VOCs行业须纳入总量精细化管理(佛环函2023 29号) [依据: 粤环发[2019]2号 ] |
| STD_062 | VOCs削减替代-顺德 | 粤环发[2019]2号  | 顺德区新增VOCs>=300kg/a须按2:1削减替代。替代方案须明确:替代项目名称/减排量/确认文件。不得仅写'由区统筹' [依据: 粤环发[2019]2号 ] |

## 7. 审核判断清单

| 审核点 | 触发条件 | 所需证据 |
|--------|---------|---------|
| 稀释比说明 | 稀释比未说明或取值无依据 | 稀释比取值说明, 稀释剂MSDS, 稀释剂用量核算 |
| 低VOCs替代论证 | 使用溶剂型原辅料但未论证替代可行性 | 低VOCs替代方案论证, 不可替代技术说明, VOCs最小化承诺 |
| VOCs总量指标 | VOCs总量未明确 | VOCs排放总量核算表, 总量指标来源文件, 佛山4+2行业归类说明 |
| VOCs削减替代-顺德 | VOCs>300kg/a未做削减替代 | VOCs削减替代方案, 替代来源确认文件, 总量审核意见 |

## 8. 判断依据与限值

- **稀释比说明**: 使用稀释剂的工序（涂布/印刷/贴合）须说明稀释比(%)和稀释剂类型。稀释剂VOCs全部计入源强 [依据: GB 31572-2015 5.2]
- **低VOCs替代论证**: 使用溶剂型胶粘剂/油墨/清洗剂的，须评估低VOCs替代方案的可行性。若不可替代，须说明技术原因并承诺VOCs排放最小化 [依据: GB 33372-2020 附录A]
- **VOCs总量指标**: 须明确VOCs有组织+无组织年排放总量。佛山4+2重点VOCs行业须纳入总量精细化管理(佛环函2023 29号) [依据: 粤环发[2019]2号 ]
- **VOCs削减替代-顺德**: 顺德区新增VOCs>=300kg/a须按2:1削减替代。替代方案须明确:替代项目名称/减排量/确认文件。不得仅写'由区统筹' [依据: 粤环发[2019]2号 ]

## 9. 输出格式

```json
{
  "skill_id": "15",
  "skill_name": "15_VOCs总量控制审核",
  "conclusion": "匹配 | 不匹配 | 部分匹配 | 无法判断",
  "key_findings": [
    {
      "review_point": "",
      "finding": "",
      "evidence_location": "",
      "law_basis": ""
    }
  ],
  "evidence_used": [],
  "law_basis_used": [],
  "review_comment": "",
  "manual_review_needed": true
}
```

### 输出说明
- `conclusion` 只能是四类：匹配/不匹配/部分匹配/无法判断
- `review_comment` 须写成经办人可直接执行的审核意见，引用具体条款号和限值
- `manual_review_needed` 默认 `true`，仅当所有审核点均"匹配"且证据链完整时可为 `false`
- 没有证据时必须输出"无法判断"，禁止编造审核结论

## 10. 人工复核提示

- 所有 `conclusion` 为"不匹配"或"无法判断"的条目 **必须** 人工复核
- 证据来源必须可追溯到报告原文的章节/页码
- 法规依据如有更新（如标准修订、地方政策调整），需同步更新本 skill 的判断阈值
- 若报告缺失关键信息（如无MSDS、无设备参数），标记为"无法判断"并指明缺失项
