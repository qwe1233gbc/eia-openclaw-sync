# 02_环评投资核算审核

## 1. 审核目标
对环评报告中环评投资核算相关内容进行独立审核，判断是否满足法规和标准要求。

## 2. 适用场景
- 佛山市塑胶行业建设项目环境影响报告表审核
- 涉VOCs、胶水/涂胶、复合/熟化工艺的项目

## 3. 不适用场景
- 纯注塑/挤塑项目（无VOCs产生环节）
- 报告书级别的环评文件

## 4. 输入材料
- 环评报告原文相关章节（MinerU解析后的Markdown）
- 可选：法规库检索结果

## 5. 需要提取的报告证据
- 编制依据章节
- 标准版本号核查表
- 能耗水耗一览表(表11格式)
- 供水/供电/供气协议

## 6. 需要检索的法规库依据
- **编制依据** (环办环评[2020]33号 ): 编制依据须包含环办环评(2020)33号、HJ884、HJ942、HJ1122、HJ819、HJ1207、粤环办(2013)31号、佛环(2021)41号、广东省工业源VOCs核算方法(2023修订版) [依据: 环办环评[2020]33号
- **能耗水耗** (环办环评[2020]33号 附件2): 须列出水、电、天然气/液化石油气等能源消耗量及用途，注明来源（市政供水/管网天然气等） [依据: 环办环评[2020]33号 附件2]
## 7. 判断逻辑
1. 从报告中提取上述证据
2. 逐条对照法规库依据
3. 判断证据是否完整、数值是否合规、逻辑是否闭合
4. 形成审核结论

## 8. 输出格式
```json
{
  "skill_id": "02",
  "skill_name": "02_环评投资核算审核",
  "conclusion": "匹配 | 不匹配 | 部分匹配 | 无法判断",
  "key_findings": [],
  "evidence_used": [],
  "law_basis_used": [],
  "review_comment": "",
  "manual_review_needed": true
}
```

## 9. 人工复核提示
- 所有conclusion为"不匹配"或"无法判断"的条目必须人工复核
- 证据来源必须可追溯到报告原文页码/章节
- 法规依据如有更新（如标准修订），需同步更新本skill
