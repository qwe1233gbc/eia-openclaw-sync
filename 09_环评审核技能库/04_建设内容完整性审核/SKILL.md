# 04_建设内容完整性审核

## 1. 审核目标
对环评报告中建设内容完整性相关内容进行独立审核，判断是否满足法规和标准要求。

## 2. 适用场景
- 佛山市塑胶行业建设项目环境影响报告表审核
- 涉VOCs、胶水/涂胶、复合/熟化工艺的项目

## 3. 不适用场景
- 纯注塑/挤塑项目（无VOCs产生环节）
- 报告书级别的环评文件

## 4. 输入材料
- 环评报告原文相关章节（MinerU解析后的Markdown）
- 可选：法规库检索结果

## 5. 需要提取的报告证据
- 项目组成一览表(表3格式)
- 总平面布置图
- 产品方案一览表(表2格式)
- 产品规格说明书
- 现有工程环评批复及验收文件
- 依托可行性分析
- 以新带老工程内容表
- 工艺流程图(含产污标识)
- 设备清单与流程图的对应表

## 6. 需要检索的法规库依据
- **项目组成完整性** (环办环评[2020]33号 附件2): 须列表说明主体/辅助/公用/环保/储运/依托工程，注明建筑面积、层数、建筑功能 [依据: 环办环评[2020]33号 附件2]
- **产品方案** (环办环评[2020]33号 附件2): 产品方案须含序号、产品名称、设计产能、规格尺寸。涉涂装/复合/印刷工序须注明涂装面积、复合面积或印刷面积 [依据: 环办环评[2020]33号 附件2]
- **改扩建项目依托关系** (环办环评[2020]33号 附件2): 改扩建/迁建/技改项目须说明与现有工程的依托关系（公辅设施/环保设施/生产设备），注明'以新带老'内容 [依据: 环办环评[2020]33号 附件2]
- **工艺流程图** (环办环评[2020]33号 附件2): 工艺流程图须标注所有产污环节（G废气/W废水/N噪声/S固废），注塑/挤出/涂布/复合/印刷/贴合/烘干/熟化各工序单独标注。图须与设备清单一一对应 [依据: 环办环评[2020]33号 附件2]
## 7. 判断逻辑
1. 从报告中提取上述证据
2. 逐条对照法规库依据
3. 判断证据是否完整、数值是否合规、逻辑是否闭合
4. 形成审核结论

## 8. 输出格式

```json
}
```

## 9. 人工复核提示
- 所有conclusion为"不匹配"或"无法判断"的条目必须人工复核
- 证据来源必须可追溯到报告原文页码/章节
- 法规依据如有更新（如标准修订），需同步更新本skill
