## 三、经验卡适用边界

### Pending 卡严格限制

| 规则 | 说明 |
|------|------|
| Pending卡不能给强判断 | 只能MEDIUM级 relevance → 仅+1分 |
| Pending卡不改变 gold answer | 不因经验卡存在而改变审核结论的强度 |
| Pending卡须标注 can_support_judgement=false | 必须显式声明"不支撑判定" |

### 行业错配硬过滤

| 规则 | 案例 |
|------|------|
| 注塑经验不适用于涂布 | EXP_07 (注塑工序) × Q16 (涂布产能) → 硬过滤丢弃 |
| 不同工艺类型的经验不可跨行业泛化 | 即使 VOCs 相关也不可跨行业 |

### Task Type 硬过滤

| 规则 | 案例 |
|------|------|
| standard_calculation 类型不咨询经验卡 | EXP_01/EXP_06/EXP_08 → Q10硬过滤 |
| 经验卡设计时已标注 not_applicable_task_types | 过滤需在 Step1 硬过滤阶段执行 |
