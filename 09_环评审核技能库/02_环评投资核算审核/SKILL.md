# 02_环评投资核算审核

## 1. 审核目标
环评投资核算审核的独立审核，判断报告中的相关内容是否满足法规、标准和指南要求。

## 2. 适用场景
- 佛山市塑胶行业建设项目环境影响报告表
- 涉VOCs、胶水/涂胶、复合/熟化工艺的项目

## 3. 不适用场景
- 报告书级别环评文件
- 非塑胶行业项目
- 纯注塑/挤塑项目（无VOCs产生环节，参照使用）

## 4. 输入材料
- 环评报告MinerU解析后的Markdown全文
- 可选：法规库检索结果（从 `03_法规库_明文依据/` 获取）
- 对应Dify工作流：`06_Dify工作流/2-环评投资核算.yml`

## 5. 文本提取策略

### 关键词评分筛选
使用全文检索，无预定义关键词权重

### 章节边界定位
- 从报告全文检索相关章节

原始工作流使用关键词评分累积 + 章节边界截断的混合策略，总分超过阈值且位于目标章节内的文本片段被提取为审核素材。

## 6. 法规库检索

审核时需从法规库检索以下模块的标准条款：

| STD_002 | 编制依据 | 环办环评[2020]33号  | 编制依据须包含环办环评(2020)33号、HJ884、HJ942、HJ1122、HJ819、HJ1207、粤环办(2013)31号、佛环(2021)41号、广东省工业源VOCs核算方法(2023修订版 |
| STD_010 | 劳动定员与工作制度 | 环办环评[2020]33号 附件2 | 须明确员工人数、生产班制（几班倒）、年生产天数、日运行时数。这些参数直接影响VOCs排放量计算和活性炭更换周期 [依据: 环办环评[2020]33号 附件2] |
| STD_011 | 能耗水耗 | 环办环评[2020]33号 附件2 | 须列出水、电、天然气/液化石油气等能源消耗量及用途，注明来源（市政供水/管网天然气等） [依据: 环办环评[2020]33号 附件2] |

## 7. 审核判断清单

| 审核点 | 触发条件 | 所需证据 |
|--------|---------|---------|
| 编制依据 | 报告引用标准版本过时 | 编制依据章节, 标准版本号核查表 |
| 劳动定员与工作制度 | 劳动定员/班制与实际不符 | 劳动定员表(表10格式), 工作制度说明 |
| 能耗水耗 | 能耗水耗数据缺失 | 能耗水耗一览表(表11格式), 供水/供电/供气协议 |

## 8. 判断依据与限值

- **编制依据**: 编制依据须包含环办环评(2020)33号、HJ884、HJ942、HJ1122、HJ819、HJ1207、粤环办(2013)31号、佛环(2021)41号、广东省工业源VOCs核算方法(2023修订版) [依据: 环办环评[2020]33号 ]
- **劳动定员与工作制度**: 须明确员工人数、生产班制（几班倒）、年生产天数、日运行时数。这些参数直接影响VOCs排放量计算和活性炭更换周期 [依据: 环办环评[2020]33号 附件2]
- **能耗水耗**: 须列出水、电、天然气/液化石油气等能源消耗量及用途，注明来源（市政供水/管网天然气等） [依据: 环办环评[2020]33号 附件2]

## 9. 输出格式

```json
{
  "skill_id": "02",
  "skill_name": "02_环评投资核算审核",
  "conclusion": "匹配 | 不匹配 | 部分匹配 | 无法判断",
  "key_findings": [
    {
      "review_point": "",
      "finding": "",
      "evidence_location": "",
      "law_basis": ""
    }
  ],
  "evidence_used": [],
  "law_basis_used": [],
  "review_comment": "",
  "manual_review_needed": true
}
```

### 输出说明
- `conclusion` 只能是四类：匹配/不匹配/部分匹配/无法判断
- `review_comment` 须写成经办人可直接执行的审核意见，引用具体条款号和限值
- `manual_review_needed` 默认 `true`，仅当所有审核点均"匹配"且证据链完整时可为 `false`
- 没有证据时必须输出"无法判断"，禁止编造审核结论

## 10. 人工复核提示

- 所有 `conclusion` 为"不匹配"或"无法判断"的条目 **必须** 人工复核
- 证据来源必须可追溯到报告原文的章节/页码
- 法规依据如有更新（如标准修订、地方政策调整），需同步更新本 skill 的判断阈值
- 若报告缺失关键信息（如无MSDS、无设备参数），标记为"无法判断"并指明缺失项
