# 13_活性炭参数审核

## 1. 审核目标
活性炭参数审核的独立审核，判断报告中的相关内容是否满足法规、标准和指南要求。

## 2. 适用场景
- 佛山市塑胶行业建设项目环境影响报告表
- 涉VOCs、胶水/涂胶、复合/熟化工艺的项目

## 3. 不适用场景
- 报告书级别环评文件
- 非塑胶行业项目
- 纯注塑/挤塑项目（无VOCs产生环节，参照使用）

## 4. 输入材料
- 环评报告MinerU解析后的Markdown全文
- 可选：法规库检索结果（从 `03_指南解析_明文标准库/` 获取）
- 对应Dify工作流：`06_Dify工作流/10-运营期间产污系数使用合理性判定.yml`

## 6. 法规库检索

审核时需从法规库检索以下模块的标准条款：

| STD_046 | 治理设施选型 | HJ 1122-2020 附录G | 塑胶行业优先采用:水喷淋+干式过滤+活性炭吸附。不得单独使用光氧催化/低温等离子/水喷淋等低效设施 [依据: HJ 1122-2020 附录G] |
| STD_047 | 预处理-水喷淋/干式过滤 | HJ 2026-2013 6.1 | 活性炭吸附前须设水喷淋降温+干式过滤除湿。进入活性炭的废气:温度<40C/湿度<80%/颗粒物<1mg/m3 [依据: HJ 2026-2013 6.1] |
| STD_048 | 活性炭-碘值和比表面积 | HJ 2026-2013 6.3.1 | 指南附件四:颗粒炭碘值>=800mg/g。佛环函(2024)70号:蜂窝炭碘值>=650mg/g。颗粒比表面积>=850m2/g,蜂窝>=750m2/g [依据: HJ 2026-2013 6.3.1 |
| STD_049 | 活性炭-过滤风速和停留时间 | HJ 2026-2013 6.3.3 | 指南附件四:颗粒炭过滤风速<0.5m/s,蜂窝<1.2m/s。停留时间>=0.5s。佛环函要求:停留时间0.5-1s [依据: HJ 2026-2013 6.3.3] |
| STD_050 | 活性炭-装填量 | HJ 2026-2013 6.3.4 | 装填量=炭箱体积×堆积密度。指南:装填厚度>=300mm。佛环函:蜂窝厚度>=600mm [依据: HJ 2026-2013 6.3.4] |

## 7. 审核判断清单

| 审核点 | 触发条件 | 所需证据 |
|--------|---------|---------|
| 治理设施选型 | 治理设施选型无依据 | 治理设施工艺流程图, 治理设施设计参数, 治理设施选型论证 |
| 预处理-水喷淋/干式过滤 | 活性炭前缺预处理设施 | 预处理设施设计参数, 温湿度监测记录, 颗粒物检测报告 |
| 活性炭-碘值和比表面积 | 活性炭碘值不达标 | 活性炭采购合同或检测报告, 碘值或比表面积检测报告 |
| 活性炭-过滤风速和停留时间 | 过滤风速/停留时间未核算或超标 | 过滤风速计算(风量/炭箱截面积), 停留时间计算, 炭箱尺寸图 |
| 活性炭-装填量 | 装填量未经计算 | 活性炭装填量计算, 炭箱尺寸图, 装填厚度测量 |
| 活性炭-更换周期 | 更换周期仅写定期 | 更换周期计算过程, 各参数取值依据, 更换记录/台账 |

## 8. 判断依据与限值

- **治理设施选型**: 塑胶行业优先采用:水喷淋+干式过滤+活性炭吸附。不得单独使用光氧催化/低温等离子/水喷淋等低效设施 [依据: HJ 1122-2020 附录G]
- **预处理-水喷淋/干式过滤**: 活性炭吸附前须设水喷淋降温+干式过滤除湿。进入活性炭的废气:温度<40C/湿度<80%/颗粒物<1mg/m3 [依据: HJ 2026-2013 6.1]
- **活性炭-碘值和比表面积**: 指南附件四:颗粒炭碘值>=800mg/g。佛环函(2024)70号:蜂窝炭碘值>=650mg/g。颗粒比表面积>=850m2/g,蜂窝>=750m2/g [依据: HJ 2026-2013 6.3.1]
- **活性炭-过滤风速和停留时间**: 指南附件四:颗粒炭过滤风速<0.5m/s,蜂窝<1.2m/s。停留时间>=0.5s。佛环函要求:停留时间0.5-1s [依据: HJ 2026-2013 6.3.3]
- **活性炭-装填量**: 装填量=炭箱体积×堆积密度。指南:装填厚度>=300mm。佛环函:蜂窝厚度>=600mm [依据: HJ 2026-2013 6.3.4]
- **治理效率取值**: 单级活性炭处理效率一般取50-75%。多级组合:总效率=1-Π(1-ηi)。不得将单级效率简单相加 [依据: 广东省工业源VOCs减排量核算方法(2023修订版) 表24; HJ 2026-2013 4.3]
- **废活性炭产生量**: 废活性炭年产生量=单次装填量×年更换次数+年吸附VOCs量。更换次数=365/更换周期(d)。须与危废合同处置能力匹配 [依据: HJ 2026-2013 附录A]
- **活性炭安全设施**: 每个炭箱须安装:压差计(压力增大至限值时提醒换炭)/温度传感器(进气<=40C,83C报警)/防火阀(65-80C自动关闭)。此要求来自佛环函(2024)70号 [依据: HJ 2026-2013 8.1]

## 9. 输出格式

```json
{
  "skill_id": "13",
  "skill_name": "13_活性炭参数审核",
  "conclusion": "匹配 | 不匹配 | 部分匹配 | 无法判断",
  "key_findings": [
    {
      "review_point": "",
      "finding": "",
      "evidence_location": "",
      "law_basis": ""
    }
  ],
  "evidence_used": [],
  "law_basis_used": [],
  "review_comment": "",
  "manual_review_needed": true
}
```

### 输出说明
- `conclusion` 只能是四类：匹配/不匹配/部分匹配/无法判断
- `review_comment` 须写成经办人可直接执行的审核意见，引用具体条款号和限值
- `manual_review_needed` 默认 `true`，仅当所有审核点均"匹配"且证据链完整时可为 `false`
- 没有证据时必须输出"无法判断"，禁止编造审核结论

## 10. 人工复核提示

- 所有 `conclusion` 为"不匹配"或"无法判断"的条目 **必须** 人工复核
- 证据来源必须可追溯到报告原文的章节/页码
- 法规依据如有更新（如标准修订、地方政策调整），需同步更新本 skill 的判断阈值
- 若报告缺失关键信息（如无MSDS、无设备参数），标记为"无法判断"并指明缺失项
