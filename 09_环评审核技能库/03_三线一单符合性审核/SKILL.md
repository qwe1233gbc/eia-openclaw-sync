# 03_三线一单符合性审核

## 1. 审核目标
三线一单符合性审核的独立审核，判断报告中的相关内容是否满足法规、标准和指南要求。

## 2. 适用场景
- 佛山市塑胶行业建设项目环境影响报告表
- 涉VOCs、胶水/涂胶、复合/熟化工艺的项目

## 3. 不适用场景
- 报告书级别环评文件
- 非塑胶行业项目
- 纯注塑/挤塑项目（无VOCs产生环节，参照使用）

## 4. 输入材料
- 环评报告MinerU解析后的Markdown全文
- 可选：法规库检索结果（从 `03_指南解析_明文标准库/` 获取）
- 对应Dify工作流：`06_Dify工作流/4-三线一单管控内容一致性判定.yml`

### 章节边界定位
- **起始标记**: 三线一单, 生态环境准入清单, 环境管控单元, 与.*三线一单.*相符性

原始工作流使用关键词评分累积 + 章节边界截断的混合策略，总分超过阈值且位于目标章节内的文本片段被提取为审核素材。

## 6. 法规库检索

审核时需从法规库检索以下模块的标准条款：

| STD_003 | 三线一单符合性 | 顺府发[2021]11号 附件 | 须逐级核对省/市/区三级三线一单生态环境分区管控方案。顺德区项目须引用顺德区三线一单方案（顺府发〔2021〕11号） [依据: 顺府发[2021]11号 附件] |
| STD_005 | 规划符合性 | 佛环[2022]3号 第三章 | 须分析与佛山市生态环境保护十四五规划、所在镇街土地利用规划的符合性 [依据: 佛环[2022]3号 第三章] |
| STD_006 | 选址合理性 | 环办环评[2020]33号 附件2 | 须说明项目用地性质（工业用地），分析周边500m范围内环境敏感点（学校/医院/居民区）分布及影响 [依据: 环办环评[2020]33号 附件2] |

## 7. 审核判断清单

| 审核点 | 触发条件 | 所需证据 |
|--------|---------|---------|
| 三线一单符合性 | 未逐级核对三线一单 | 省/市/区三级三线一单管控单元查询截图, 管控单元准入清单对照表 |
| 规划符合性 | 未分析相关规划符合性 | 规划文件相关条款摘录, 规划图件（用地规划图） |
| 选址合理性 | 选址周边有环境敏感点 | 土地证/用地规划许可证, 周边敏感点分布图, 敏感点距厂界距离测量 |

## 8. 判断依据与限值

- **三线一单符合性**: 须逐级核对省/市/区三级三线一单生态环境分区管控方案。顺德区项目须引用顺德区三线一单方案（顺府发〔2021〕11号） [依据: 顺府发[2021]11号 附件]
- **规划符合性**: 须分析与佛山市生态环境保护十四五规划、所在镇街土地利用规划的符合性 [依据: 佛环[2022]3号 第三章]
- **选址合理性**: 须说明项目用地性质（工业用地），分析周边500m范围内环境敏感点（学校/医院/居民区）分布及影响 [依据: 环办环评[2020]33号 附件2]

## 9. 输出格式

```json
{
  "skill_id": "03",
  "skill_name": "03_三线一单符合性审核",
  "conclusion": "匹配 | 不匹配 | 部分匹配 | 无法判断",
  "key_findings": [
    {
      "review_point": "",
      "finding": "",
      "evidence_location": "",
      "law_basis": ""
    }
  ],
  "evidence_used": [],
  "law_basis_used": [],
  "review_comment": "",
  "manual_review_needed": true
}
```

### 输出说明
- `conclusion` 只能是四类：匹配/不匹配/部分匹配/无法判断
- `review_comment` 须写成经办人可直接执行的审核意见，引用具体条款号和限值
- `manual_review_needed` 默认 `true`，仅当所有审核点均"匹配"且证据链完整时可为 `false`
- 没有证据时必须输出"无法判断"，禁止编造审核结论

## 10. 人工复核提示

- 所有 `conclusion` 为"不匹配"或"无法判断"的条目 **必须** 人工复核
- 证据来源必须可追溯到报告原文的章节/页码
- 法规依据如有更新（如标准修订、地方政策调整），需同步更新本 skill 的判断阈值
- 若报告缺失关键信息（如无MSDS、无设备参数），标记为"无法判断"并指明缺失项
