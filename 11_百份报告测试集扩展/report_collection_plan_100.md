# 百份塑胶行业环评报告测试集扩展计划

## 1. 当前状态

当前已有约 17 份塑胶行业环评报告，已用于初步 seed_qa 构建和人工标注流程验证。

## 2. 扩展目标

逐步扩展到约 100 份报告，形成更稳定的实验测试集。

## 3. 报告来源

1. 已有顺德历史项目资料；
2. 佛山市 / 顺德区生态环境公开信息；
3. 受理公告、拟审批公示、批复文件；
4. 同类塑胶行业涉 VOCs 项目；
5. 含修改意见或批注的报告优先。

## 4. 样本筛选重点

优先选择：

- 涉胶水 / 涂胶 / 复合 / 熟化；
- 涉 VOCs 废气收集治理；
- 涉活性炭吸附；
- 涉废活性炭和危废识别；
- 有修改意见、批注、终稿或批复链条。

## 5. seed_qa 生成建议

每份报告生成 5—8 道题，避免过多增加人工复核成本。
