# 05_环境质量现状数据审核

## 1. 审核目标
环境质量现状数据审核的独立审核，判断报告中的相关内容是否满足法规、标准和指南要求。

## 2. 适用场景
- 佛山市塑胶行业建设项目环境影响报告表
- 涉VOCs、胶水/涂胶、复合/熟化工艺的项目

## 3. 不适用场景
- 报告书级别环评文件
- 非塑胶行业项目
- 纯注塑/挤塑项目（无VOCs产生环节，参照使用）

## 4. 输入材料
- 环评报告MinerU解析后的Markdown全文
- 可选：法规库检索结果（从 `03_指南解析_明文标准库/` 获取）
- 对应Dify工作流：`06_Dify工作流/6-环境质量现状数据引用是否有误.yml`

## 5. 文本提取策略

### 关键词评分筛选
| 关键词组 | 权重 |
|---------|------|
| `区域环境质量现状` | 28 |
| `环境质量现状` | 24 |
| `现状监测` | 22 |
| `引用.*数据` | 20 |
| `监测时间` | 16 |
| `监测点位` | 16 |
| `监测因子` | 16 |
| `达标` | 10 |

### 章节边界定位
- **起始标记**: 区域环境质量现状, 环境质量现状
- **结束标记**: 环境保护目标, 评价适用标准, 主要环境影响

原始工作流使用关键词评分累积 + 章节边界截断的混合策略，总分超过阈值且位于目标章节内的文本片段被提取为审核素材。

## 6. 法规库检索

审核时需从法规库检索以下模块的标准条款：

| STD_027 | 大气环境现状 | GB 3095-2012  | 须引用佛山市生态环境局公布的最近一年环境质量公报数据。顺德区项目优先引用顺德区生态环境状况公报。项目周边5km范围内近3年监测数据须注明来源 [依据: GB 3095-2012 ] |
| STD_028 | 地表水环境现状 | GB 3838-2002  | 引用佛山市主要水环境控制单元水质数据或最近环境质量公报。纳污水体须有最近1年的监测数据 [依据: GB 3838-2002 ] |
| STD_029 | 声环境现状 | GB 3096-2008  | 厂界周边50m范围内有声环境敏感点时须进行声环境现状监测，昼夜各1次 [依据: GB 3096-2008 ] |

## 7. 审核判断清单

| 审核点 | 触发条件 | 所需证据 |
|--------|---------|---------|
| 大气环境现状 | 大气环境现状数据来源不明或过时 | 环境质量公报引用页面, 周边监测数据来源及时间 |
| 地表水环境现状 | 地表水现状数据缺失 | 水环境质量公报, 纳污水体监测报告 |
| 声环境现状 | 声环境现状未监测 | 声环境现状监测报告, 声环境功能区划文件 |

## 8. 判断依据与限值

- **大气环境现状**: 须引用佛山市生态环境局公布的最近一年环境质量公报数据。顺德区项目优先引用顺德区生态环境状况公报。项目周边5km范围内近3年监测数据须注明来源 [依据: GB 3095-2012 ]
- **地表水环境现状**: 引用佛山市主要水环境控制单元水质数据或最近环境质量公报。纳污水体须有最近1年的监测数据 [依据: GB 3838-2002 ]
- **声环境现状**: 厂界周边50m范围内有声环境敏感点时须进行声环境现状监测，昼夜各1次 [依据: GB 3096-2008 ]

## 9. 输出格式

```json
{
  "skill_id": "05",
  "skill_name": "05_环境质量现状数据审核",
  "conclusion": "匹配 | 不匹配 | 部分匹配 | 无法判断",
  "key_findings": [
    {
      "review_point": "",
      "finding": "",
      "evidence_location": "",
      "law_basis": ""
    }
  ],
  "evidence_used": [],
  "law_basis_used": [],
  "review_comment": "",
  "manual_review_needed": true
}
```

### 输出说明
- `conclusion` 只能是四类：匹配/不匹配/部分匹配/无法判断
- `review_comment` 须写成经办人可直接执行的审核意见，引用具体条款号和限值
- `manual_review_needed` 默认 `true`，仅当所有审核点均"匹配"且证据链完整时可为 `false`
- 没有证据时必须输出"无法判断"，禁止编造审核结论

## 10. 人工复核提示

- 所有 `conclusion` 为"不匹配"或"无法判断"的条目 **必须** 人工复核
- 证据来源必须可追溯到报告原文的章节/页码
- 法规依据如有更新（如标准修订、地方政策调整），需同步更新本 skill 的判断阈值
- 若报告缺失关键信息（如无MSDS、无设备参数），标记为"无法判断"并指明缺失项
