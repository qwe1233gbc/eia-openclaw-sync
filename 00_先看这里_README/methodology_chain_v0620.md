# 研究方法链（2026-06-20修订）

## 方法链

```
公开资料与内部样本收集
        ↓
塑胶行业涉VOCs样本筛选
        ↓
样本来源分级与证据可追溯性核验   ← 替代原"完整样本链匹配"
        ↓
报告文本解析与证据片段抽取
        ↓
明文标准库构建（71条标准卡，17个审核模块）
        ↓
审核工作流规则整理（16个工作流）
        ↓
类案经验卡抽取与relevance过滤（8条active + 5条candidate）
        ↓
seed QA / benchmark构建（18题主实验 + 2题备选）
        ↓
证据链审计与人工复核
        ↓
A/B/C对照实验验证（报告only / +标准库 / +标准+经验）
        ↓
标准—经验双库迭代更新
```

## 样本来源分级

| 等级 | 定义 | 当前状态 | 允许用途 |
|------|------|---------|---------|
| A类完整链 | 同一项目：受理+修改+终稿+批复 | 当前数据集无A类样本 | case_study, benchmark |
| B类半链 | 同一项目：2-3类材料 | 康明等ai_package项目 | experience_card, seed_qa |
| C类单点 | 仅有1类材料 | 磁盘批复PDF | standard_card, candidate |

## 与旧方法链的区别

| 旧 | 新 |
|----|----|
| 受理公告—修改意见—终稿—批复样本链匹配 | 样本来源分级与证据可追溯性核验 |
| 要求四段式完整链 | ABC三级分级管理 |
| 缺链项目视为不合格 | 不同等级承担不同研究用途 |
| 声称样本链完整 | 如实标注当前无A类完整链 |
