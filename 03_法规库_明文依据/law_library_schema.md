# 法规库字段结构

为降低人工维护成本，法规库条目字段控制在 8 个以内。

| 字段 | 中文说明 | 填写要求 |
|---|---|---|
| 法规编号 | 条目唯一编号 | 可按 LAW_001 编号 |
| 法规名称 | 标准、指南或规范名称 | 必须准确 |
| 条款位置 | 条款号、章节、表号或页码 | 能追溯即可 |
| 适用行业/工艺/污染物 | 说明适用范围 | 如 C292、涂胶、VOCs、废活性炭 |
| 审核要求 | 条款要求的核心内容 | 用审核语言概括 |
| 报告所需证据 | 报告中需要检查的证据 | 如原辅材料表、废气章节、危废表 |
| 可支撑的技能 | 可服务哪些 skill | 填 skill 名称或编号 |
| 备注 | 不确定性或人工复核说明 | 可为空 |
