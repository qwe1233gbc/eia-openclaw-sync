# 14_危险废物识别审核

## 1. 审核目标
危险废物识别审核的独立审核，判断报告中的相关内容是否满足法规、标准和指南要求。

## 2. 适用场景
- 佛山市塑胶行业建设项目环境影响报告表
- 涉VOCs、胶水/涂胶、复合/熟化工艺的项目

## 3. 不适用场景
- 报告书级别环评文件
- 非塑胶行业项目
- 纯注塑/挤塑项目（无VOCs产生环节，参照使用）

## 4. 输入材料
- 环评报告MinerU解析后的Markdown全文
- 可选：法规库检索结果（从 `03_指南解析_明文标准库/` 获取）
- 对应Dify工作流：`06_Dify工作流/8-污染物排放标准内容判断.yml`

## 5. 文本提取策略

### 关键词评分筛选
使用全文检索，无预定义关键词权重

### 章节边界定位
- 从报告全文检索相关章节

原始工作流使用关键词评分累积 + 章节边界截断的混合策略，总分超过阈值且位于目标章节内的文本片段被提取为审核素材。

## 6. 法规库检索

审核时需从法规库检索以下模块的标准条款：

| STD_056 | 一般固废 | GB 18599-2020 4.1 | 一般固废(废边角料/废包装/次品)须分类列出产生量。废塑料边角料可破碎后回用或外售综合利用 [依据: GB 18599-2020 4.1] |
| STD_057 | 危险废物-识别 | GB 18597-2001 4.1; HJ 2025-2012 4.1 | 除废活性炭(HW49)外须识别:废胶水桶(HW49)/废机油(HW08)/废溶剂桶(HW49)/含胶废抹布(HW49)/废切削液(HW09)/废水处理污泥(HW49) [依据: GB 18597-20 |
| STD_058 | 危险废物-贮存 | GB 18597-2001 6.3 | 危废贮存须防风/防雨/防晒/防渗漏。液态危废设围堰或导流沟。不同危废分区存放，设危废识别标志。贮存面积须满足产生量周转需求 [依据: GB 18597-2001 6.3] |
| STD_059 | 危险废物-处置 | HJ 2025-2012 5.1 | 各类危废须分别委托有资质单位处置。危废处置合同的处置能力须>=危废年产生量。须附危废转移联单 [依据: HJ 2025-2012 5.1] |

## 7. 审核判断清单

| 审核点 | 触发条件 | 所需证据 |
|--------|---------|---------|
| 一般固废 | 一般固废产生量未核算 | 一般固废产生量核算, 一般固废处置/利用合同, 贮存场所照片 |
| 危险废物-识别 | 危废识别不全 | 危废产生源识别表, 危废代码核对表 |
| 危险废物-贮存 | 危废暂存间设计不规范 | 危废贮存场所照片, 防渗设计, 危废识别标志照片 |
| 危险废物-处置 | 危废处置合同缺失或能力不足 | 各类危废委托处置合同, 危废转移联单, 危废台账 |

## 8. 判断依据与限值

- **一般固废**: 一般固废(废边角料/废包装/次品)须分类列出产生量。废塑料边角料可破碎后回用或外售综合利用 [依据: GB 18599-2020 4.1]
- **危险废物-识别**: 除废活性炭(HW49)外须识别:废胶水桶(HW49)/废机油(HW08)/废溶剂桶(HW49)/含胶废抹布(HW49)/废切削液(HW09)/废水处理污泥(HW49) [依据: GB 18597-2001 4.1; HJ 2025-2012 4.1]
- **危险废物-贮存**: 危废贮存须防风/防雨/防晒/防渗漏。液态危废设围堰或导流沟。不同危废分区存放，设危废识别标志。贮存面积须满足产生量周转需求 [依据: GB 18597-2001 6.3]
- **危险废物-处置**: 各类危废须分别委托有资质单位处置。危废处置合同的处置能力须>=危废年产生量。须附危废转移联单 [依据: HJ 2025-2012 5.1]

## 9. 输出格式

```json
{
  "skill_id": "14",
  "skill_name": "14_危险废物识别审核",
  "conclusion": "匹配 | 不匹配 | 部分匹配 | 无法判断",
  "key_findings": [
    {
      "review_point": "",
      "finding": "",
      "evidence_location": "",
      "law_basis": ""
    }
  ],
  "evidence_used": [],
  "law_basis_used": [],
  "review_comment": "",
  "manual_review_needed": true
}
```

### 输出说明
- `conclusion` 只能是四类：匹配/不匹配/部分匹配/无法判断
- `review_comment` 须写成经办人可直接执行的审核意见，引用具体条款号和限值
- `manual_review_needed` 默认 `true`，仅当所有审核点均"匹配"且证据链完整时可为 `false`
- 没有证据时必须输出"无法判断"，禁止编造审核结论

## 10. 人工复核提示

- 所有 `conclusion` 为"不匹配"或"无法判断"的条目 **必须** 人工复核
- 证据来源必须可追溯到报告原文的章节/页码
- 法规依据如有更新（如标准修订、地方政策调整），需同步更新本 skill 的判断阈值
- 若报告缺失关键信息（如无MSDS、无设备参数），标记为"无法判断"并指明缺失项
