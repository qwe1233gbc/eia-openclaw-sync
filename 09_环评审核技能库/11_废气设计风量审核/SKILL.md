# 11_废气设计风量审核

## 1. 审核目标
废气设计风量审核的独立审核，判断报告中的相关内容是否满足法规、标准和指南要求。

## 2. 适用场景
- 佛山市塑胶行业建设项目环境影响报告表
- 涉VOCs、胶水/涂胶、复合/熟化工艺的项目

## 3. 不适用场景
- 报告书级别环评文件
- 非塑胶行业项目
- 纯注塑/挤塑项目（无VOCs产生环节，参照使用）

## 4. 输入材料
- 环评报告MinerU解析后的Markdown全文
- 可选：法规库检索结果（从 `03_法规库_明文依据/` 获取）
- 对应Dify工作流：`06_Dify工作流/18-废气收集风量与设计风量核算.yml`

### 章节边界定位
- **起始标记**: 运营期, 营运期, 废气, 主要环境影响和保护措施
- **结束标记**: 废水自行监测计划, 声环境, 固体废物, 环境保护措施监督检查清单

原始工作流使用关键词评分累积 + 章节边界截断的混合策略，总分超过阈值且位于目标章节内的文本片段被提取为审核素材。

## 6. 法规库检索

审核时需从法规库检索以下模块的标准条款：

| STD_044 | 风量核算-包围型集气罩 | HJ 2026-2013 6.2.1 | 包围型集气罩排风量Q=(10x2+F)Vx(m3/s)。x=罩口距污染源距离(m),F=罩口面积(m2),Vx=控制风速(m/s)。有挡板取完整值,无挡板取0.75倍 [依据: HJ 2026-201 |
| STD_045 | 风量-多罩口并联 | HJ 2026-2013 6.2.5 | 多罩口并联时总风量须考虑管道阻力平衡和漏风系数(一般取10-20%)。各支管须设风量调节阀 [依据: HJ 2026-2013 6.2.5] |

## 7. 审核判断清单

| 审核点 | 触发条件 | 所需证据 |
|--------|---------|---------|
| 风量核算-包围型集气罩 | 风量计算缺罩口面积或罩距 | 罩口尺寸实测, 罩距x取值, 风量核算过程 |
| 风量-多罩口并联 | 多罩口并联风量简单相加 | 罩口风量分配表, 管道阻力计算, 风量调节阀设置方案 |

## 8. 判断依据与限值

- **风量-多罩口并联**: 多罩口并联时总风量须考虑管道阻力平衡和漏风系数(一般取10-20%)。各支管须设风量调节阀 [依据: HJ 2026-2013 6.2.5]

## 9. 输出格式

```json
{
  "skill_id": "11",
  "skill_name": "11_废气设计风量审核",
  "conclusion": "匹配 | 不匹配 | 部分匹配 | 无法判断",
  "key_findings": [
    {
      "review_point": "",
      "finding": "",
      "evidence_location": "",
      "law_basis": ""
    }
  ],
  "evidence_used": [],
  "law_basis_used": [],
  "review_comment": "",
  "manual_review_needed": true
}
```

### 输出说明
- `conclusion` 只能是四类：匹配/不匹配/部分匹配/无法判断
- `review_comment` 须写成经办人可直接执行的审核意见，引用具体条款号和限值
- `manual_review_needed` 默认 `true`，仅当所有审核点均"匹配"且证据链完整时可为 `false`
- 没有证据时必须输出"无法判断"，禁止编造审核结论

## 10. 人工复核提示

- 所有 `conclusion` 为"不匹配"或"无法判断"的条目 **必须** 人工复核
- 证据来源必须可追溯到报告原文的章节/页码
- 法规依据如有更新（如标准修订、地方政策调整），需同步更新本 skill 的判断阈值
- 若报告缺失关键信息（如无MSDS、无设备参数），标记为"无法判断"并指明缺失项
