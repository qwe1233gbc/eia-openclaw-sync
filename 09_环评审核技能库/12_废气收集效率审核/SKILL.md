# 12_废气收集效率审核

## 1. 审核目标
对环评报告中废气收集效率相关内容进行独立审核，判断是否满足法规和标准要求。

## 2. 适用场景
- 佛山市塑胶行业建设项目环境影响报告表审核
- 涉VOCs、胶水/涂胶、复合/熟化工艺的项目

## 3. 不适用场景
- 纯注塑/挤塑项目（无VOCs产生环节）
- 报告书级别的环评文件

## 4. 输入材料
- 环评报告原文相关章节（MinerU解析后的Markdown）
- 可选：法规库检索结果

## 5. 需要提取的报告证据
- 收集效率取值依据(表23)
- 控制风速实测/计算
- 收集方式与收集效率对应关系

## 6. 需要检索的法规库依据
- **收集效率取值** (广东省工业源VOCs减排量核算方法(2023修订版) 表23): 全密闭负压:95%;包围型(控制风速>=0.5m/s):80%;包围型(控制风速0.3-0.5m/s):60%;外部型(>=0.5m/s):40%;无措施:0%。不得将包围型取95% [依据: 广东省工业源VOCs减排量核算方法(2023修
## 7. 判断逻辑
1. 从报告中提取上述证据
2. 逐条对照法规库依据
3. 判断证据是否完整、数值是否合规、逻辑是否闭合
4. 形成审核结论

## 8. 输出格式

```json
{

  "conclusion": "匹配 | 不匹配 | 部分匹配 | 无法判断",
  "key_findings": [],
  "evidence_used": [],
  "law_basis_used": [],
  "review_comment": "",
  "manual_review_needed": true
}
```

## 9. 人工复核提示
- 所有conclusion为"不匹配"或"无法判断"的条目必须人工复核
- 证据来源必须可追溯到报告原文页码/章节
- 法规依据如有更新（如标准修订），需同步更新本skill
