# 09_源强定量核算审核

## 1. 审核目标
源强定量核算审核的独立审核，判断报告中的相关内容是否满足法规、标准和指南要求。

## 2. 适用场景
- 佛山市塑胶行业建设项目环境影响报告表
- 涉VOCs、胶水/涂胶、复合/熟化工艺的项目

## 3. 不适用场景
- 报告书级别环评文件
- 非塑胶行业项目
- 纯注塑/挤塑项目（无VOCs产生环节，参照使用）

## 4. 输入材料
- 环评报告MinerU解析后的Markdown全文
- 可选：法规库检索结果（从 `03_法规库_明文依据/` 获取）
- 对应Dify工作流：`06_Dify工作流/10-运营期间产污系数与定量核算判定.yml`

## 5. 文本提取策略

### 关键词评分筛选
使用全文检索，无预定义关键词权重

### 章节边界定位
- 从报告全文检索相关章节

原始工作流使用关键词评分累积 + 章节边界截断的混合策略，总分超过阈值且位于目标章节内的文本片段被提取为审核素材。

审核时需从法规库检索以下模块的标准条款：

| STD_017 | 胶粘剂用量-设备匹配 | 佛山市塑胶行业环评编制指南(试行) 四(二)3 | 须按设备型号对照表9核算单台设备的原料用量系数，验证总用量与设备产能是否匹配 [依据: 佛山市塑胶行业环评编制指南(试行) 四(二)3] |
| STD_039 | VOCs物料衡算 | 广东省工业源VOCs减排量核算方法(2023修订版) 公式 | 须做VOCs物料衡算:从原辅料VOCs投入→各工序产生→有组织收集→无组织逸散→治理设施削减→排放，全链条闭合。有组织/无组织分别列表 [依据: 广东省工业源VOCs减排量核算方法(2023修订版)  |

## 7. 审核判断清单

| 审核点 | 触发条件 | 所需证据 |
|--------|---------|---------|
| 胶粘剂用量核算-涂胶量 | 涂胶量取值无依据 | 涂胶量计算过程, C/V/d/P各参数取值依据, 复合工艺类型确认 |
| 胶粘剂用量-设备匹配 | 原辅料用量与设备产能不匹配 | 设备/原料用量匹配核算表, 设备技术规格书 |
| VOCs物料衡算 | VOCs产生量无物料衡算 | VOCs物料衡算表, 有组织/无组织VOCs核算表 |

## 8. 判断依据与限值

- **胶粘剂用量-设备匹配**: 须按设备型号对照表9核算单台设备的原料用量系数，验证总用量与设备产能是否匹配 [依据: 佛山市塑胶行业环评编制指南(试行) 四(二)3]
- **VOCs物料衡算**: 须做VOCs物料衡算:从原辅料VOCs投入→各工序产生→有组织收集→无组织逸散→治理设施削减→排放，全链条闭合。有组织/无组织分别列表 [依据: 广东省工业源VOCs减排量核算方法(2023修订版) 公式]

## 9. 输出格式

```json
{
  "skill_id": "09",
  "skill_name": "09_源强定量核算审核",
  "conclusion": "匹配 | 不匹配 | 部分匹配 | 无法判断",
  "key_findings": [
    {
      "review_point": "",
      "finding": "",
      "evidence_location": "",
      "law_basis": ""
    }
  ],
  "evidence_used": [],
  "law_basis_used": [],
  "review_comment": "",
  "manual_review_needed": true
}
```

### 输出说明
- `conclusion` 只能是四类：匹配/不匹配/部分匹配/无法判断
- `review_comment` 须写成经办人可直接执行的审核意见，引用具体条款号和限值
- `manual_review_needed` 默认 `true`，仅当所有审核点均"匹配"且证据链完整时可为 `false`
- 没有证据时必须输出"无法判断"，禁止编造审核结论

## 10. 人工复核提示

- 所有 `conclusion` 为"不匹配"或"无法判断"的条目 **必须** 人工复核
- 证据来源必须可追溯到报告原文的章节/页码
- 法规依据如有更新（如标准修订、地方政策调整），需同步更新本 skill 的判断阈值
- 若报告缺失关键信息（如无MSDS、无设备参数），标记为"无法判断"并指明缺失项
