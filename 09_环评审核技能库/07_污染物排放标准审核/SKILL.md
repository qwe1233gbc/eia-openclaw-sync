# 07_污染物排放标准审核

## 1. 审核目标
污染物排放标准审核的独立审核，判断报告中的相关内容是否满足法规、标准和指南要求。

## 2. 适用场景
- 佛山市塑胶行业建设项目环境影响报告表
- 涉VOCs、胶水/涂胶、复合/熟化工艺的项目

## 3. 不适用场景
- 报告书级别环评文件
- 非塑胶行业项目
- 纯注塑/挤塑项目（无VOCs产生环节，参照使用）

## 4. 输入材料
- 环评报告MinerU解析后的Markdown全文
- 可选：法规库检索结果（从 `03_指南解析_明文标准库/` 获取）
- 对应Dify工作流：`06_Dify工作流/8-污染物排放标准内容判断.yml`

## 5. 文本提取策略

### 关键词评分筛选
使用全文检索，无预定义关键词权重

### 章节边界定位
- 从报告全文检索相关章节

原始工作流使用关键词评分累积 + 章节边界截断的混合策略，总分超过阈值且位于目标章节内的文本片段被提取为审核素材。

## 7. 审核判断清单

| 审核点 | 触发条件 | 所需证据 |
|--------|---------|---------|
| 废气排放标准-VOCs | VOCs排放标准引用过时(DB44/815/814) | 排放标准适用性分析表, 排气筒监测数据, 标准版本确认 |
| 废气排放标准-颗粒物 | 颗粒物排放限值引用错误 | 颗粒物排放核算, 厂界监测数据 |
| 废气排放标准-臭气 | 臭气浓度排放标准遗漏 | 臭气浓度排放核算, 排气筒高度 |
| 废水排放标准 | 废水排放标准引用错误 | 废水水质检测报告, 纳管协议, 污水处理厂接纳标准 |
| 噪声排放标准 | 噪声功能区类别不确定 | 声环境功能区划文件, 厂界噪声监测报告 |
| 非甲烷总烃折算 | 非甲烷总烃与VOCs的换算关系不明确 | VOCs组分分析, 非甲烷总烃折算说明 |

## 8. 判断依据与限值

- **废气排放标准-臭气**: 臭气浓度有组织排放根据排气筒高度执行GB14554-93表1限值:15m<=2000(无量纲)/25m<=6000;厂界无组织20(无量纲) [依据: GB 14554-93 表2]
- **废水排放标准**: 生活污水排入城镇污水处理厂:COD<=500/BOD5<=300/SS<=400/氨氮--(DB44/26-2001第二时段三级)。生产废水直接排放须执行更严标准 [依据: DB44/26-2001 表4]
- **噪声排放标准**: 厂界噪声按声环境功能区:3类昼<=65/夜<=55;2类昼<=60/夜<=50。具体分区须依据顺德区声环境功能区划 [依据: GB 12348-2008 表1]
- **非甲烷总烃折算**: 若MSDS中VOCs含量以总挥发份表示，须将非甲烷总烃单独标注。非甲烷总烃排放浓度须满足60mg/m3限值(DB44/2367-2022) [依据: DB44/2367-2022 5.2]

## 9. 输出格式

```json
{
  "skill_id": "07",
  "skill_name": "07_污染物排放标准审核",
  "conclusion": "匹配 | 不匹配 | 部分匹配 | 无法判断",
  "key_findings": [
    {
      "review_point": "",
      "finding": "",
      "evidence_location": "",
      "law_basis": ""
    }
  ],
  "evidence_used": [],
  "law_basis_used": [],
  "review_comment": "",
  "manual_review_needed": true
}
```

### 输出说明
- `conclusion` 只能是四类：匹配/不匹配/部分匹配/无法判断
- `review_comment` 须写成经办人可直接执行的审核意见，引用具体条款号和限值
- `manual_review_needed` 默认 `true`，仅当所有审核点均"匹配"且证据链完整时可为 `false`
- 没有证据时必须输出"无法判断"，禁止编造审核结论

## 10. 人工复核提示

- 所有 `conclusion` 为"不匹配"或"无法判断"的条目 **必须** 人工复核
- 证据来源必须可追溯到报告原文的章节/页码
- 法规依据如有更新（如标准修订、地方政策调整），需同步更新本 skill 的判断阈值
- 若报告缺失关键信息（如无MSDS、无设备参数），标记为"无法判断"并指明缺失项
