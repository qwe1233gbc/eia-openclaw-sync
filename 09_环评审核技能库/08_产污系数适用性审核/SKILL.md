# 08_产污系数适用性审核

## 1. 审核目标
产污系数适用性审核的独立审核，判断报告中的相关内容是否满足法规、标准和指南要求。

## 2. 适用场景
- 佛山市塑胶行业建设项目环境影响报告表
- 涉VOCs、胶水/涂胶、复合/熟化工艺的项目

## 3. 不适用场景
- 报告书级别环评文件
- 非塑胶行业项目
- 纯注塑/挤塑项目（无VOCs产生环节，参照使用）

## 4. 输入材料
- 环评报告MinerU解析后的Markdown全文
- 可选：法规库检索结果（从 `03_法规库_明文依据/` 获取）
- 对应Dify工作流：`06_Dify工作流/9-运营期间产污系数使用合理性判断.yml`

## 5. 文本提取策略

### 关键词评分筛选
使用全文检索，无预定义关键词权重

### 章节边界定位
- 从报告全文检索相关章节

原始工作流使用关键词评分累积 + 章节边界截断的混合策略，总分超过阈值且位于目标章节内的文本片段被提取为审核素材。

## 6. 法规库检索

审核时需从法规库检索以下模块的标准条款：

| STD_035 | VOCs源强-注塑/挤出 | HJ 1122-2020 表22 | 注塑(塑料包装及容器)VOCs产污系数2.70kg/t-产品。挤出(塑料板/管/型材)VOCs 1.50。须注明系数来源(指南表22/HJ1122) [依据: HJ 1122-2020 表22] |
| STD_036 | VOCs源强-涂布/复合 | HJ 1122-2020 表22 | 涂布/复合工序VOCs=胶粘剂年用量×VOCs含量(%)×(1-固含量%)。须根据胶水MSDS中的VOCs含量和固含量分别核算 [依据: HJ 1122-2020 表22] |
| STD_037 | VOCs源强-印刷 | HJ 1122-2020 表22 | 印刷工序VOCs=油墨年用量×VOCs含量(%)+清洗剂年用量×VOCs含量(%)。参照《塑料包装印刷VOCs实用手册》取油墨和清洗剂参数 [依据: HJ 1122-2020 表22] |
| STD_038 | VOCs源强-烘干/熟化 | HJ 1122-2020 表22 | 烘干和熟化工序仍残留溶剂挥发，须纳入VOCs产生源。按胶水VOCs产生量的5-15%估算残留挥发量 [依据: HJ 1122-2020 表22] |

## 7. 审核判断清单

| 审核点 | 触发条件 | 所需证据 |
|--------|---------|---------|
| VOCs源强-注塑/挤出 | 注塑VOCs产生量计算方式错误或系数取值错误 | VOCs源强核算表, 产品类型与系数对应关系, 同类项目类比数据 |
| VOCs源强-涂布/复合 | 涂布/复合工序VOCs未单独核算 | 胶水VOCs含量(MSDS), 胶水年用量统计, 涂布/复合VOCs核算表 |
| VOCs源强-印刷 | 印刷工序VOCs遗漏 | 油墨/清洗剂MSDS, 油墨/清洗剂年用量, 印刷VOCs核算表 |
| VOCs源强-烘干/熟化 | 烘干/熟化工序VOCs遗漏 | 烘干/熟化温度和时间参数, 残留挥发量估算依据 |

## 8. 判断依据与限值

- **VOCs源强-注塑/挤出**: 注塑(塑料包装及容器)VOCs产污系数2.70kg/t-产品。挤出(塑料板/管/型材)VOCs 1.50。须注明系数来源(指南表22/HJ1122) [依据: HJ 1122-2020 表22]
- **VOCs源强-涂布/复合**: 涂布/复合工序VOCs=胶粘剂年用量×VOCs含量(%)×(1-固含量%)。须根据胶水MSDS中的VOCs含量和固含量分别核算 [依据: HJ 1122-2020 表22]
- **VOCs源强-烘干/熟化**: 烘干和熟化工序仍残留溶剂挥发，须纳入VOCs产生源。按胶水VOCs产生量的5-15%估算残留挥发量 [依据: HJ 1122-2020 表22]

## 9. 输出格式

```json
{
  "skill_id": "08",
  "skill_name": "08_产污系数适用性审核",
  "conclusion": "匹配 | 不匹配 | 部分匹配 | 无法判断",
  "key_findings": [
    {
      "review_point": "",
      "finding": "",
      "evidence_location": "",
      "law_basis": ""
    }
  ],
  "evidence_used": [],
  "law_basis_used": [],
  "review_comment": "",
  "manual_review_needed": true
}
```

### 输出说明
- `conclusion` 只能是四类：匹配/不匹配/部分匹配/无法判断
- `review_comment` 须写成经办人可直接执行的审核意见，引用具体条款号和限值
- `manual_review_needed` 默认 `true`，仅当所有审核点均"匹配"且证据链完整时可为 `false`
- 没有证据时必须输出"无法判断"，禁止编造审核结论

## 10. 人工复核提示

- 所有 `conclusion` 为"不匹配"或"无法判断"的条目 **必须** 人工复核
- 证据来源必须可追溯到报告原文的章节/页码
- 法规依据如有更新（如标准修订、地方政策调整），需同步更新本 skill 的判断阈值
- 若报告缺失关键信息（如无MSDS、无设备参数），标记为"无法判断"并指明缺失项
