# 08_产污系数适用性审核

## 1. 审核目标
对环评报告中产污系数适用性相关内容进行独立审核，判断是否满足法规和标准要求。

## 2. 适用场景
- 佛山市塑胶行业建设项目环境影响报告表审核
- 涉VOCs、胶水/涂胶、复合/熟化工艺的项目

## 3. 不适用场景
- 纯注塑/挤塑项目（无VOCs产生环节）
- 报告书级别的环评文件

## 4. 输入材料
- 环评报告原文相关章节（MinerU解析后的Markdown）
- 可选：法规库检索结果

## 5. 需要提取的报告证据
- VOCs源强核算表
- 产品类型与系数对应关系
- 同类项目类比数据
- 胶水VOCs含量(MSDS)
- 胶水年用量统计
- 涂布/复合VOCs核算表
- 油墨/清洗剂MSDS
- 油墨/清洗剂年用量
- 印刷VOCs核算表
- 烘干/熟化温度和时间参数
- 残留挥发量估算依据

## 6. 需要检索的法规库依据
- **VOCs源强-注塑/挤出** (HJ 1122-2020 表22): 注塑(塑料包装及容器)VOCs产污系数2.70kg/t-产品。挤出(塑料板/管/型材)VOCs 1.50。须注明系数来源(指南表22/HJ1122) [依据: HJ 1122-2020 表22]
- **VOCs源强-涂布/复合** (HJ 1122-2020 表22): 涂布/复合工序VOCs=胶粘剂年用量×VOCs含量(%)×(1-固含量%)。须根据胶水MSDS中的VOCs含量和固含量分别核算 [依据: HJ 1122-2020 表22]
- **VOCs源强-印刷** (HJ 1122-2020 表22): 印刷工序VOCs=油墨年用量×VOCs含量(%)+清洗剂年用量×VOCs含量(%)。参照《塑料包装印刷VOCs实用手册》取油墨和清洗剂参数 [依据: HJ 1122-2020 表22]
- **VOCs源强-烘干/熟化** (HJ 1122-2020 表22): 烘干和熟化工序仍残留溶剂挥发，须纳入VOCs产生源。按胶水VOCs产生量的5-15%估算残留挥发量 [依据: HJ 1122-2020 表22]
## 7. 判断逻辑
1. 从报告中提取上述证据
2. 逐条对照法规库依据
3. 判断证据是否完整、数值是否合规、逻辑是否闭合
4. 形成审核结论

## 9. 人工复核提示
- 所有conclusion为"不匹配"或"无法判断"的条目必须人工复核
- 证据来源必须可追溯到报告原文页码/章节
- 法规依据如有更新（如标准修订），需同步更新本skill
