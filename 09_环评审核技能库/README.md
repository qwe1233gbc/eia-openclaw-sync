# 环评审核技能库

本目录是当前课题的核心建设对象之一：将原有 Dify 工作流、法规/标准依据、样本证据和类案经验，整理成可独立执行、可组合调用、可人工复核的审核 skill。

## 设计原则

- 一个 skill 只审核一个独立条目；
- 各 skill 互不依赖；
- 每个 skill 可单独执行；
- 总 skill 只汇总，不替代单项判断；
- 所有输出统一结构，便于后续知识条件对照实验。

## 技能库与法规库的关系

```text
报告证据 + 法规/标准依据 + skill 审核流程 + 人工复核边界 → 单项审核结果
```

法规/标准依据库提供“依据是什么”，skill 提供“怎么审、检查什么、如何输出”。类案经验只作为相关 skill 的辅助提示，不能替代法规/标准依据。

## 与 Dify 工作流的关系

Dify 工作流是 skill 的来源之一，但不是最终组织单位。当前整理目标是：

- 从工作流中抽取稳定审核步骤；
- 为每个审核模块补齐输入、判断依据、输出字段和复核边界；
- 将类案经验绑定到具体 skill，而不是作为泛化规则单独使用；
- 保留人工复核要求，尤其是证据不足、适用范围不清或跨材料矛盾的任务。
