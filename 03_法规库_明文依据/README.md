# 法规库（明文依据库）

## 定位

法规库是环评智能审核系统的底层依据库，回答"依据是什么、限值/公式/参数是什么、报告应提供哪些证据"。

法规库不直接替代审核判断，而是供各 skill 检索和引用。

## 来源

- 《佛山市塑胶行业建设项目环评文件编制技术参考指南（试行）》
- 国家标准（GB 37822/33372/31572/38508/3095/3096/3838/12348/18597/18599/14554/16297/41616 等）
- 环保行业标准（HJ 2026/1122/1207/819/884/169/2025/942 等）
- 广东省地方标准（DB44/2367/27/26/814/815）
- 生态环境部/广东省/佛山市政策文件
- 建设项目环境影响报告表编制技术指南（污染影响类）（试行）

## 数据来源

本法规库基于原 `03_指南解析_明文标准库/` 中的 71 条标准卡 + 26 个标准全文解析构建。

## 核心字段

| 字段 | 说明 |
|------|------|
| 法规编号 | 如 STD_001 |
| 法规名称/审核模块 | 如 "废气收集效率取值" |
| 条款位置 | 标准编号 + 条款号 |
| 适用条件 | 触发该审核点的条件 |
| 核心要求 | 法规/标准的关键限值或判断规则 |
| 报告所需证据 | 报告中应提供的支撑材料 |
| 可支撑的技能 | 对应 09_环评审核技能库 中的哪些 skill |

## 使用方式

每个审核 skill 从法规库中检索相关条目，引用其条款号和限值，
结合报告证据片段形成结构化审核输出。
